# Cat Energy

Разметка шаблона "Cat Energy" от HTML Academy.

## Сборка

Используйте систему потоковой сборки [Gulp](https://gulpjs.com/), чтобы собрать
Cat Energy:

```bash
gulp build
```

## Запуск

Чтобы посмотреть проект в работе, выполните следующую команду:

```bash
npm start
```
